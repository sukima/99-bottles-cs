99 Bottles of Beer

Sung by CoffeeScript
Written by Devin Weaver

This is an example application written in CoffeeScript to sing the verses to the song "99 Bottles of Beer (On the Wall)". It is intended as a candidate for the 99 Bottles of Beer programming database site.

Documentation

The documentation is split into three modules:

  • 99bottles.coffee - This is the main application logic. It is responsible for looping through the song and outputting the text with an appropriate display method.
  • runner.coffee - This is the implementation specific code which sets up the page. It is the normal cruft you find in the HTML but in this case it was modularized to it's own file.
  • collapsible.coffee - This is a custom jQuery plugin to allow a div to collapse. (click on the introduction to see this work)

Click on a module above to view the detailed annotations to the source code.
